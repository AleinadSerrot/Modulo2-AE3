# Modulo2-AE3
módulo 2 actividad AE 3

Instrucciones
1) Personalizar con css: cambiar el color de fondo de la pagina, aplicar diferentes colores al texto del encabezado, párrafo y pie de página, utilizar fuente sans-serif para el contenido

2) Modelo de cajas:agregar bordes, relleno(padding) y margenes(margin) a los elementos principales encabezado y secciones, definir un ancho máximo de a página y centrar el contenido en la pantalla

3) Utilizar media queries para ajustar el diseño en pantallas pequeñas (ancho menor a 768px): cambiar tamaño del texto para que sea legible, ajustar los márgenes y rellenos para maximizar el uso del espacio en pantalla, asegurar que la imagen sea responsiva y se adapte al tamaño del dispositivo
